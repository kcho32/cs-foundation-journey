# CS Foundation Journey

## Goal

Build strong computer science fundamentals so I can explain not only how to implement software, but also why certain designs, data structures, and technologies are chosen.

## Rules

- Weekdays: 0.5 ~ 1hour of focused study
- Weekends: 3–5 hours for review and implementation
- Write a short learning log every day
- Focus on understanding trade-offs, not just memorizing definitions

## Study Principle

For every topic, answer these questions:

1. What problem does this solve?
2. How does it work internally?
3. What are the trade-offs?
4. When should I use it?
5. When should I avoid it?
